# French Tutor Agent

A simple exploration into OpenAI's speech-to-speech API, with the end goal that I might use this agent to practice speaking French to.

## Dependencies
* uv 0.7.8
* OpenAPI API key in the `OPENAI_API_KEY` env var

## Installation

1. > uv venv

2. > uv pip install
